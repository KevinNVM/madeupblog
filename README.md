## About madeupblog
Blogging Website Built With Laravel 9.51.0

## Stacks Used
1. Laravel
2. InertiaJS
3. VueJS

## Features
1. Create, Read, Update, and Delete (Manage) Posts
2. Manage posts categories
3. User can leave a comment on a post
4. User can leave a like for a post
5. User have the ability to copy document from another document editor software
